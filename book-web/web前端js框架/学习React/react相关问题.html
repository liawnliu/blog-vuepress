# create-react-app 问题

create-react-app 这个脚手架在 3.3.0 到 4.0.0 有内存占用过大和启动慢的问题解决办法：

  1. 打开项目的 node_modules/react-scripts/config/webpack.config.js
  2. 在new ESLintPlugin({下面一行加上cache: true,，参考9886issue (opens new window)
  3. 在将// TypeScript type checking与new ESLintPlugin({之间的代码注释掉，也就是禁止使用ForkTsCheckerWebpackPlugin，参考8096issue (opens new window)，该 issue 也有其他改法也可以试试
